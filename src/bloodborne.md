# 血源诅咒 8/10

**时间：2024-03-02 19:49:01**

白了，血源历险记结束  考虑到游戏发售的年代可以给8/10，
优点:
①场景非常的不错，虽然看得我眼睛疼
②枪反和虚血配合的很好
③地图设计好，绕一圈绕回老家是非常爽的
④动作武器很帅而且带变形
缺点:
①也是地图，疑似嗯推这个开捷径跑图导致提灯放得特别少，个别boss要跑很远，中间的路程还堆一大堆怪比如该隐赫斯特的鬼叫女和旧亚楠的狼人，鬼叫女虽然打不到我但是叫的我耳朵疼，狼人我又跑不过
②血瓶和子弹要刷
③没有坐火，想让怪刷新必须回梦境
④没搞懂地牢是玩什么的，完了没会员的话想白金还得从头开始刷地牢，而且还很折磨
⑤不能洗点，没点奥术想玩猎人道具都玩不了
⑥boss设计是真烂，还是喜欢只狼那种每个boss做得都很好的游戏，这游戏有些boss看名字都想不起来招式，只有科斯孤儿 渴血怪兽 玛丽亚修女 神父 路德维希 该隐赫斯特的那个boss算是设计的好的，噩梦之主之类的就是纯屎 当然跟只狼比boss设计有些赖皮了

这些个老游戏侧重的疑似是推图探索不是boss战，还是让我的资历只停留在环狼源吧
不过该说不说这游戏的尬吹粉疑似有点太多了，烂的都能吹成好的，信了你天下第一的邪
