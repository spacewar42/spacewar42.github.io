# 死亡搁浅 7/10

**时间：2025-04-04 19:18:51**

死亡搁浅 7/10
20年朋友就推荐过，下了个盗版玩不下去，后来epic送了，玩一半又忘了为什么不玩了，这两天闲的没事下回来结果三天就通关，果然玩这类慢节奏游戏得看眼缘啊，说不定哪天也会觉得荒野大镖客2之类的游戏好玩了。
之前的7分打得太随意了很多游戏其实值8分以上的，但这个游戏是确确实实的7分，像看了一部不知名导演拍的文艺片，打完后短暂感受到了很久没感受过的空虚感
